Replaces the author link with your Google+ Profile Link or adds the Special Link in your blogpost

== Description ==

We've built another easy and free wordpress plugin. It will show ones author information  (profile picture and name) in the Google search results.

= German =

* Erstellen Sie ein Google+ Profil (http://plus.google.com)
* Loggen Sie sich bei Google+ ein
* Klicken Sie hier um Ihr Profil zu öffnen: http://plus.google.com/me
* Finden Sie Ihren eigenen Google+ Profillink indem Sie Ihre eigenen Profilseite besuchen. Kopieren Sie dann den entsprechenden Link in die Zwischenablage (siehe Bild unten)
* Aktivieren Sie dieses Plugin.
* Wechseln Sie zur Blog-Profilseite und fügen Sie den gerade kopierten Google+ Profillink in das Feld "Google Plus Link"
* Gehen Sie in Einstellungen > Google Plus Author und legen Sie alle nötigen Einstellungen fest (siehe unten)
* Folgen Sie dieser Anleitung um Ihr Google+-Profil mit Ihrem Blog zu verbinden: http://support.google.com/webmasters/bin/answer.py?hl=de&answer=1408986
* Testen Sie Ihre Seite. Benutzen Sie dazu das Rich Snippets Testing Tool (http://www.google.com/webmasters/tools/richsnippets)
* Wenn Sie Ihr Profilbild sehen können ist alles OK. Wenn nicht, haben Sie etwas falsch gemacht.
* Seien Sie sich im klaren, dass es keine Garantie dafür gibt, dass Rich Snippets in den Suchergebnissen bei Google angezeigt werden.
    
= English =
* Create a Google+ Profile (http://plus.google.com/Plus)
* Login to Google+
* Click here to open your profile: http://plus.google.com/me
* Find your Google+ Profile link by visiting your own page and copy the link (shown and marked in the image below) to clipboard
* Activate this plugin.
* Go to your profile page and paste your profile link where it says "Google Plus Link"
* Go to Settings > Google Plus Author and make changes if needed (this page - see below)
* Follow this instructions to link your Google+ Profile with your blog: http://support.google.com/webmasters/bin/answer.py?hl=en&answer=1408986
* Test your site using the Rich Snippets Testing Tool (http://www.google.com/webmasters/tools/richsnippets)
* If you can see your profile picture everything is okay. If not, you have made something wrong.
* Note that there is no guarantee that a Rich Snippet will be shown for your page on actual search results!
